Dappy

Dappy or Draw And Paint in PYthon. A GPL drawing program written in Python, PyGTK, and PyCairo. Dappy was forked from the painthon project on Google code.

Download this project as a .zip file Download this project as a tar.gz file

Dappy is far from complete but is almost usable.

Project goals:

  • A functional paint program written in Python.
  • A modular program that can be loaded into other Python applications.
